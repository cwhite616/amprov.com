# Amprov

Everyone can hear now.

This repository contains the dependency-free static marketing website for the
Amprov MVP. It is ready to publish with GitHub Pages.

## Preview locally

```sh
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Publish

In the GitHub repository settings, open **Pages**, choose **Deploy from a
branch**, and publish the root of `main`.

The privacy policy is available at `/privacy.html`.
